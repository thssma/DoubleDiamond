DoubleDiamond Original App - Deploy sem Lovable

Arquivos principais:
- index.html
- script.js
- style.css
- manifest.json
- service-worker.js
- _redirects

Como publicar no GitHub:
1. Abra o repositório https://github.com/thssma/DoubleDiamond
2. Faça upload/substituição destes arquivos na raiz do repositório.
3. Confirme o commit.

Como publicar no Netlify:
1. Se o Netlify já está conectado ao GitHub, ele fará deploy automático.
2. Caso contrário, vá em Netlify > Add new site > Import from Git.
3. Build command: deixe vazio.
4. Publish directory: deixe vazio ou use .
5. Deploy.

Depois abra:
https://SEU-SITE.netlify.app/?v=original-app

Observação:
Este pacote usa o aplicativo original completo, não o /dd/mvp.html do Lovable.
