DoubleDiamond Role-Based Login V1

Inclui em um único deploy:
1. Página inicial de login para cliente.
2. Acesso discreto para Owner/Employee pelo ícone 💎.
3. Perfis locais: client, employee, owner.
4. Botão Voltar e botão Sair.
5. CRM renomeado de Empresa/Companies para Clientes.
6. Correções pequenas de schema para field_signatures, automation_flow_runs e ai_insights.
